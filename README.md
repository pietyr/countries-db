# countries-db

[Live version](https://countries-db.kisiel.io/)

## Prerequisites

- node 24+
- npm 11+

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
